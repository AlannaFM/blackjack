<h1 align="center">♠️ Blackjack Multiplayer RMI</h1>

###

<p align="left">
O Blackjack Multiplayer RMI é uma versão multiplayer do clássico jogo de cartas Blackjack (21), desenvolvida utilizando Java RMI para permitir a comunicação entre jogadores em diferentes máquinas. O objetivo do jogo é alcançar uma pontuação próxima de 21 sem ultrapassá-la, competindo contra outros jogadores em tempo real. O projeto foi desenvolvido com foco em programação distribuída, comunicação cliente-servidor e lógica de jogos multiplayer.
</p>

###

### ➤ O que eu aprendi

* **Java RMI** - Comunicação remota entre cliente e servidor.
* **Programação Cliente-Servidor** - Gerenciamento das conexões dos jogadores.
* **Sockets e Comunicação em Rede** - Troca de informações em tempo real.
* **Lógica de Jogo** - Implementação das regras do Blackjack.

###

<h3 align="left">➤ Tecnologias Utilizadas</h3>

* **Java** - Desenvolvimento da lógica do jogo e sistema distribuído.
* **Java RMI** - Comunicação remota entre os jogadores.
* **POO** - Organização e estruturação do projeto.

###

<h2 align="left">🕹️ Como Jogar:</h2>

* Inicie o servidor do jogo.
* Execute os clientes e conecte os jogadores ao servidor.
* O objetivo é atingir a pontuação mais próxima de 21 sem ultrapassar.
